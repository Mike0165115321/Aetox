# 📗 AetoxClaw: Central Core Documentation
**Unified Guide for the Lightweight Agentic System**

ยินดีต้อนรับสู่เอกสารกลางของ AetoxClaw เวอร์ชันปัจจุบัน เอกสารฉบับนี้รวบรวมทุกสิ่งที่คุณต้องรู้เกี่ยวกับสถาปัตยกรรมใหม่ที่เน้นความเร็วและความคล่องตัว

---

## 1. ปรัชญาการออกแบบ (Design Philosophy)

AetoxClaw ถูกออกแบบใหม่ภายใต้แนวคิด **"Intelligence without Bloat"** (ฉลาดโดยไม่บวม) โดยยึดหลักการ 3 ข้อ:
1.  **Local-First:** รันบนเครื่องของผู้ใช้ 100% ไม่ส่งข้อมูลออกภายนอก
2.  **Stateless Agent:** เอเจนต์เป็นเพียง "ผู้ประมวลผล" ไม่ต้องแบกภาระความจำถาวร
3.  **Context Injection:** ส่งเฉพาะข้อมูลที่จำเป็นให้เอเจนต์ในจังหวะที่ถูกต้อง

---

## 2. ระบบการจัดการบริบท (Context Management)

นี่คือจุดเปลี่ยนสำคัญของระบบเดิม (RAG) มาสู่ระบบปัจจุบัน:

### 🔄 กลไกการฉีดประวัติ (History Injection)
ระบบจะแบ่งการส่งประวัติออกเป็น 2 รูปแบบตามโหมดการทำงาน:

| โหมดการทำงาน | ข้อมูลที่ส่งให้เอเจนต์ | ประโยชน์ |
|-------------|-------------------|---------|
| **Chat Mode** | ประวัติการคุยล่าสุด 3-5 ชุด | คุยต่อเนื่องได้โดยไม่เปลืองทรัพยากร |
| **Plan Mode** | ผลลัพธ์ก้าวที่แล้ว + สรุปภาพรวมแผน | ทำงานซับซ้อนหลายขั้นตอนได้แม่นยำ |

---

## 3. เอเจนต์และบทบาท (Agents & Roles)

*   **Planner (MainAgent):** สมองส่วนหน้า ทำหน้าที่รับคำสั่งและแตกงานเป็นชิ้นเล็กๆ
*   **Orchestrator (Dispatcher):** หัวใจของระบบ ทำหน้าที่คุมจังหวะการรันงานและส่งต่อประวัติ
*   **Executor (The Doer):** มือเท้าของระบบ ทำหน้าที่ใช้เครื่องมือต่างๆ ตามบริบทที่ได้รับ
*   **Critic (The Auditor):** ผู้ตรวจสอบคุณภาพงานและให้คำแนะนำเมื่อเกิดข้อผิดพลาด

---

## 4. เครื่องมือและความสามารถ (Capabilities)

ระบบมีเครื่องมือมาตรฐานที่พร้อมใช้งาน:
- 📁 **File Manager:** จัดการไฟล์, ย้าย, ลบ, แก้ไขชื่อ, จัดระเบียบโฟลเดอร์
- 🌐 **Web Scraper:** อ่านเนื้อหาจากเว็บและสรุปใจความสำคัญ
- 👁️ **Aetox Vision:** วิเคราะห์ไฟล์เอกสาร PDF, Word และรูปภาพ (ผ่าน OCR/Parsing)
- ⚙️ **System Control:** ควบคุมการทำงานของระบบผ่าน Command Line (ภายใต้ความปลอดภัย)

---

## 5. การขยายระบบ (Extensibility)

คุณสามารถเพิ่มเครื่องมือใหม่ได้ง่ายๆ โดย:
1. สร้าง Class ใหม่ใน `aetox/tools/` ที่สืบทอดจาก `BaseTool`
2. ลงทะเบียนใน `aetox/tools/registry.py`
3. ระบบจะนำเครื่องมือใหม่ไปบอกเอเจนต์โดยอัตโนมัติ (Zero Configuration)

---

## 6. มาตรฐานความปลอดภัย (Safety Standards)

AetoxClaw ให้ความสำคัญกับความปลอดภัยสูงสุด:
- **Sandbox:** จำกัดพื้นที่ทำงานของเอเจนต์
- **Approval:** ขออนุญาตผู้ใช้ก่อนทำรายการที่สำคัญ
- **Privacy:** ข้อมูลความจำทั้งหมด (Session) อยู่ใน RAM และจะหายไปเมื่อปิดโปรแกรม (หรือล้างสถานะ)
