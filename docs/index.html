ติดตั้ง

สองทาง เลือกที่ถนัด — ทั้งคู่ได้แอปเดียวกัน

01ตัวติดตั้ง (แนะนำ)

  1. ดาวน์โหลด aetox-desktop-amd64-installer.exe
  2. ดับเบิลคลิก — ถ้า SmartScreen เตือน: More info → Run anyway
  3. ตัวติดตั้งจัดการ WebView2 + Tesseract ภาษาไทย ให้เอง
  4. เปิดจาก Start Menu → เลือก provider หรือ Ollama → คุยได้เลย

02Scoop (สาย dev)

ได้ portable build พร้อม shortcut อัปเดตผ่าน scoop update ตามปกติ

ใช้กับโมเดลอะไร? ใส่ API key ของ provider ที่คุณมี — หรือไม่ต้องมีเลยสักตัว: ติดตั้ง Ollama / LM Studio แล้วรันโมเดล open-source ฟรีในเครื่อง Aetox ทำหน้าที่ขยายศักยภาพให้โมเดลที่คุณเลือก ไม่ใช่บังคับว่าต้องใช้ค่ายไหน
ข้อมูลอยู่ไหน? ในเครื่องคุณ การคุยวิ่งตรงหาผู้ให้บริการโมเดลที่คุณเลือกเท่านั้น ไม่มี server กลางของ Aetox ไม่มี telemetry ไม่มี subscription — จ่ายเฉพาะ token ที่ใช้จริง หรือศูนย์บาทถ้ารัน local

เครื่องมือในตัว 22 ตัว
