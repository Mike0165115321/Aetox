# UX: ระหว่างรอโมเดลตอบ (Aetox CLI)

## การแสดงผล `กำลังคิด...`

- โหมดแชตเมื่อผู้ใช้ส่งข้อความสนทนา จะเริ่มแสดงสถานะ:
  - `Aetox: กำลังคิด` + อนิเมชันสัญลักษณ์หมุน
- เมื่อโมเดลเริ่มส่งข้อความกลบ (`chunk`) แรก สถานะนี้จะหายทันที และเปลี่ยนเป็นบรรทัดตอบปกติของ `Aetox:`
- ถ้าผลตอบกลับไม่รองรับสตรีม (fallback เป็นแบบไม่สตรีม) สถานะแสดงก่อนหน้านี้จะคงอยู่จนกว่าคำตอบครบแล้วจึงแสดงผลลัพธ์เดียว
- คำสั่งสกิล (`/help`, `list`, `time`, `shell` ฯลฯ) จะไม่ใช้ spinner นี้ เพราะเป็นผลลัพธ์จากเครื่องมือภายใน ไม่ใช่ LLM generation

## สถานที่ใช้งานในโค้ด

- โค้ดหลัก: `internal/app/app.go`
- ฟังก์ชันที่เพิ่มขึ้น: `startThinkingIndicator`
- จุดเรียกใช้งาน: ลูปใน `RunInteractive` ตอนก่อนเรียก `runCommandWithStream`

## โทนสีหลักของโปรเจกต์ (ANSI 256 สี)

| ตัวแปร | ค่า ANSI |
|---|---|
| `ansiBrandDark` | `\x1b[38;5;31m` |
| `ansiBrandMid` | `\x1b[38;5;45m` |
| `ansiBrandLight` | `\x1b[38;5;87m` |
| `ansiBrandBright` | `\x1b[38;5;117m` |
| `ansiText` | `\x1b[97m` |
| `ansiSubtle` | `\x1b[38;5;249m` |
| `ansiEraseLine` | `\x1b[2K\r` |

หมายเหตุ: เมื่อรันในเทอร์มินัลที่รองรับ ANSI อันนี้จะเห็นเฉดสีน้ำเงิน-ฟ้าคล้ายไทเทเนียมเป็นเอกลักษณ์ (Brand palette 4 ระดับ) และน้ำเงินขาวอ่านง่ายสำหรับข้อความหลัก.
