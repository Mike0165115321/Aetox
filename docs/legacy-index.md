# Legacy Documentation Index

This repository intentionally preserves legacy AetoxOS/AetoxClaw documents as
architecture evidence for the Go rewrite.

## Root Legacy Docs

- `AetoxClaw.md`: full technical specification for the earlier Python direction.
- `AetoxClawEcosystem.md`: ecosystem and local/hybrid/cloud concept notes.
- `เอกสารทำดา.md`: legacy Thai project notes.
- `ภาพรวม.txt`: legacy overview notes.

## Existing Docs Folder

- `docs/architecture.md`: stateless core and history injection.
- `docs/สถาปัตยกรรมระบบ (System Architecture).md`: Thai architecture note.
- `docs/algorithms_and_flowcharts.md`: orchestration, planning, memory, safety,
  and tool registry flows.
- `docs/aetox_os_core.md`: core system summary.
- `docs/Complexity.md`: resource and complexity notes.
- `docs/โครงสร้างโปรเจกต์ (Project Structure)ใ.md`: old Python project structure.
- `docs/กรณีทั่วไป.md`: general complexity notes.

## Migrated Legacy Docs
