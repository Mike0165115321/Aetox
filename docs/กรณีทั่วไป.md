
ผู้ใช้ส่งคำสั่ง → ตอบกลับ
          ↓
1. Intent Extraction:     O(L)        (L = ความยาวคำสั่ง)
2. Tool Selection:        O(1)        (Dict lookup)
3. Tool Execution:        O(T)        (T = เวลาของ tool นั้นๆ)
   ├─ Web Scraping:       O(H)        (H = ขนาด HTML)
   ├─ File Operation:     O(F)        (F = จำนวนไฟล์/ความลึกโฟลเดอร์)
   └─ LLM Generation:     O(G)        (G = เวลา inference)
4. Critic Evaluation:     O(C)        (C = เวลา LLM ตรวจสอบ)
5. Memory Update:         O(1)        (Append + Save JSON)
          ↓
รวม: O(L + T + G + C) ≈ O(G)  (เพราะ LLM inference ใช้เวลานานสุด)
